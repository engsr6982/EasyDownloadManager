# Qt6 & C++20 的多线程下载器 EasyDownloadManager

## 技术栈

1. Qt6 作为核心 UI 框架
2. C++20 作为核心编程语言
3. LibCurl 作为核心网络库 (openssl/zlib/http2)
4. SQLite3 作为核心数据库

## 特性：

### 1. URL Scheme

统一 URL Scheme 地址前缀: `edm://`

#### 新建任务

- `edm://add`

```ts
declare interface EdmCreateDownloadTask {
    url: string; // 下载链接
    method: "GET" | "POST"; // 请求方法
    fileName: string; // 下载文件名
    mimeType?: string; // MIME 类型
    userAgent?: string; // 用户代理
    pageUrl?: string; // 来源页面 URL (发起下载请求时的页面）
    pageTitle?: string; // 来源页面标题
    browser?: string; // 来源浏览器
    // headers
    cookie?: string; // 本次请求附加的 cookie
    origin?: string; // 源站点 (根）
    referer?: string; // 防盗链
}
```

### 2. 软件全局设置

每个任务创建时默认使用这里的配置进行下载

- [ ] 下载线程数(连接数)
- [ ] 带宽限制 (kb/s)
- [ ] 用户代理 (User-Agent)
- [ ] 保存路径 (默认 Downloads 文件夹)
- [ ] 临时文件路径 (默认系统临时文件夹)
- [ ] 自启动 (静默启动)
- [ ] 下载完成是否弹出完成对话框
- [ ] 代理配置(Http/Https/Socks)

### 3. 窗口

#### 3.1. 主窗口

- [x] 窗口采用上下布局(Vertical Layout)
- [x] 上部分为工具栏(新建任务、恢复任务、停止任务、删除任务、清除所有任务、设置)
- [x] 下半部分右侧为TaskList(文件、大小、状态、带宽、剩余时间、最后尝试)
- [x] TaskList支持：列排序、自定义列宽
- [ ] 下半部分左侧为文件类别 TreeView，对右侧列表进行过滤
- [ ] TreeView 为子窗口，可独立关闭(隐藏)
- [x] TreeView 提供4个顶层类别：所有任务、进行中、已完成、已取消
- [x] TreeView 每个顶层类别都有子类别：视频、音频、压缩文件、文档、可执行程序、其它
- [ ] TreeView 提供一个最高级别的文件名过滤器（正则）
- [ ]  TaskList 如果对条目进行双击操作，则打开 [任务信息窗口](#36-任务信息窗口)
- [ ]  TaskList 条目右键菜单：暂停/恢复、重新下载、[显示窗口](#35-下载任务窗口)、打开、打开文件夹、删除、彻底删除、属性

#### 3.4. 任务完成窗口

- [ ] 显示任务状态：文件名称、耗时、文件大小
- [ ] 提供：打开文件夹、打开文件、关闭

#### 3.5. 任务下载中窗口

- [ ] 上半部分为 Tab 组件
- [ ] Tab[0]: 任务信息：任务URL、状态、文件大小、已下载(含进度百分比)、带宽、剩余时间、断点续传
- [ ] Tab[1]: 任务配置(临时)：带宽限制、线程调整、是否显示完成对话框、使用代理服务器
- [ ] Tab 下面为总进度条、总进度条下为分段进度条
- [ ] 底部两个按钮 “暂停/继续”、“取消”
- [ ] 标题栏标题为 “正在下载: {文件名}”
- [ ] 标题栏禁用最大化、默认关闭行为重定向为隐藏任务窗口到[系统托盘](#4-系统托盘)(不暂停任务)

#### 3.6. 任务信息窗口

- [ ] 最上方显示任务名称(文件名)
- [ ] 下方依次显示：文件大小、状态、保存路径、临时路径、URL、来源页面网址、来源页面标题、来源浏览器、创建时间、最后尝试
- [ ] 提供：“打开日志文件”、“开开下载文件夹”、“打开页面网址”、“关闭”

### 4. 系统托盘

- [x] 单击打开主窗口

- [x] 托盘菜单固定：打开主窗口、退出程序
