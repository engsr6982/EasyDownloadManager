# Qt6 & C++20 的多线程下载器 EasyDownloadManager

## 特性：

### 1. URL Scheme

统一 URL Scheme 地址前缀: `edm://`

#### 新建任务

- `edm://add`

```ts
declare interface EDMCreateDownloadTask {
    url: string; // http(s) 协议
    threads?: number; // 本次任务使用的线程数量
    proxy?: boolean; // 本次任务是否使用代理
    limit?: number; // 本次任务带宽限制 (kb/s)
    webTitle?: string; // 页面标题
    webSiteUrl?: string; // 页面网址
    browser?: string; // 来源浏览器
}
```

### 2. 软件全局设置

每个任务创建时默认使用这里的配置进行下载

1. 下载线程数(连接数)
2. 带宽限制 (kb/s)
3. 用户代理 (User-Agent)
4. 保存路径 (默认 Downloads 文件夹)
5. 临时文件路径 (默认系统临时文件夹)
6. 自启动 (静默启动)
7. 软件启动自动恢复未完成的任务(不含失败任务)
8. 通过 URL Scheme 新建任务时，是否弹出创建下载任务对话框(二次确认)
9. 下载完成是否弹出完成对话框
10. 代理配置(Http/Https/Socks)

### 3. 窗口

#### 3.1. 主窗口

1. 窗口采用上下布局(Vertical Layout)
2. 上部分为工具栏(新建任务、恢复任务、停止任务、删除任务、清除所有任务、设置)
3. 下半部分右侧为TaskList(文件、大小、状态、带宽、剩余时间、最后尝试)
4. TaskList支持：列排序、自定义列宽、显示隐藏列
5. 下半部分左侧为文件类别 TreeView，对右侧列表进行过滤
6. TreeView 为子窗口，可独立关闭(隐藏)
7. TreeView 提供4个顶层类别：所有任务、进行中、已完成、已取消
8. TreeView 每个顶层类别都有子类别：视频、音频、压缩文件、文档、可执行程序、其它
9. TreeView 提供一个最高级别的文件名过滤器（正则）
10. TaskList 如果对条目进行双击操作，则打开 [任务信息窗口](#36-任务信息窗口)
11. TaskList 条目右键菜单：暂停/恢复、重新下载、[显示窗口](#35-下载任务窗口)、打开、打开文件夹、删除、彻底删除、属性

#### 3.2. 新建任务窗口

触发：[URL Scheme-新建任务](#1-url-scheme) / 主窗口工具栏新建任务按钮

布局:

1. 任务 URL 地址输入框 (URL Scheme自动填入)
2. 保存路径(默认引用软件设置)
3. 线程数(默认引用软件设置)
4. 带宽限制(默认引用软件设置)
5. 使用代理下载本次任务(默认引用软件设置)
6. 底部 “开始任务(下载)”、“取消” 按钮

#### 3.3. [设置窗口](#2-软件全局设置)

#### 3.4. 任务完成窗口

1. 显示任务状态：文件名称、耗时、文件大小
2. 提供：打开文件夹、打开文件、关闭

#### 3.5. 下载任务窗口

1. 上半部分为 Tab 组件
2. Tab[0]: 任务信息：任务URL、状态、文件大小、已下载(含进度百分比)、带宽、剩余时间、断点续传
3. Tab[1]: 任务配置(临时)：带宽限制、线程调整、是否显示完成对话框、使用代理服务器
4. Tab 下面为总进度条、总进度条下为分段进度条
5. 底部两个按钮 “暂停/继续”、“取消”
6. 标题栏标题为 “正在下载: {文件名}”
7. 标题栏禁用最大化、默认关闭行为重定向为隐藏任务窗口到[系统托盘](#4-系统托盘)(不暂停任务)

#### 3.6. 任务信息窗口

1. 最上方显示任务名称(文件名)
2. 下方依次显示：文件大小、状态、保存路径、临时路径、URL、来源页面网址、来源页面标题、来源浏览器、创建时间、最后尝试
3. 提供：“打开日志文件”、“开开下载文件夹”、“打开页面网址”、“关闭”

### 4. 系统托盘

1. 单击/双击打开主窗口
2. 右键显示托盘菜单
3. 托盘菜单固定：打开主窗口、暂停所有任务、取消所有任务、启动所有暂停的任务
4. 托盘菜单在固定的项下，显示进行中的任务（一个任务一项，最大显示8个），点击唤起[下载任务窗口](#35-下载任务窗口)
